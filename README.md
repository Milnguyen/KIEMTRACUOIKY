Quản Lý Danh Bạ (Contacts Management)

Mô tả dự án
Ứng dụng Quản lý danh bạ được xây dựng bằng Java với IntelliJ cho phép người dùng quản lý thông tin liên lạc bao gồm:
Số điện thoại
Nhóm danh bạ
Họ và tên
Giới tính
Địa chỉ
Ngày sinh
Email

Các tính năng chính:
1.Xem danh sách: Hiển thị danh bạ theo từng nhóm 5 mục một lần, nhấn Enter để xem tiếp.
2.Thêm mới: Nhập thông tin danh bạ, kiểm tra định dạng
Số điện thoại: 10 số, bắt đầu bằng 0
Email: hợp lệ theo chuẩn
Ngày sinh: đúng định dạng yyyy-MM-dd
3.Cập nhật: Sửa thông tin danh bạ theo số điện thoại, kiểm tra định dạng tương tự như thêm mới.
4.Xóa: Xóa danh bạ theo số điện thoại, yêu cầu xác nhận trước khi xóa.
5.Tìm kiếm: Tìm gần đúng theo số điện thoại hoặc họ tên.
6.Đọc từ file CSV: Tải danh bạ từ file data/contacts.csv. Người dùng sẽ nhận cảnh báo trước khi xóa dữ liệu cũ trong bộ nhớ.
7.Ghi vào file CSV: Lưu toàn bộ danh bạ hiện tại vào file data/contacts.csv.
8.Thoát: Thoát chương trình.

Công nghệ sử dụng
Java 8+
IntelliJ IDEA
File CSV để lưu trữ dữ liệu
List để quản lý danh bạ trong bộ nhớ
